# Modern React Blackjack

A fully responsive, modern Blackjack game built with React and Vite.

## Features

- **Modern UI**: Clean design with casino-style aesthetics and smooth animations.
- **Responsive**: Works seamlessly on desktop, tablet, and mobile.
- **Game Logic**: Complete Blackjack rules including Hit, Stand, Double Down, and Dealer logic (stand on 17).
- **State Management**: Uses React Hooks for efficient state handling.

## Getting Started

1.  **Install Dependencies**:

    ```bash
    npm install
    ```

2.  **Run Development Server**:

    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Technologies

- React
- Vite
- Vanilla CSS (Modern Variables & Animations)
